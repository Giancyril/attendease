TASKFLOW - Task Management Dashboard

A modern, full-stack task management application with authentication, real-time
filtering, and a responsive sidebar navigation system.

================================================================================
                        PROJECT DESCRIPTION
================================================================================

TaskFlow is a production-ready web application that enables users to manage
their tasks efficiently with a beautiful, intuitive interface. Built with
Next.js 16, React 19, and PostgreSQL, TaskFlow provides:

- User authentication with secure password hashing
- Complete CRUD operations for task management
- Real-time statistics and task filtering
- Responsive design with dark theme
- Professional sidebar navigation
- Multi-status task tracking (Todo, In Progress, Completed, On Hold)
- Priority levels (Low, Medium, High, Urgent)
- Mobile-friendly interface
